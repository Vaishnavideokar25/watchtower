How It Works

WatchTower detects meaningful webpage changes and helps users act on them intelligently.

Simple 5-Step Workflow

1

Secure login

Users authenticate using Firebase Authentication.

2

Add a webpage

Users enter a URL and keywords they want to track.

3

Backend processing (Python)

A Python backend fetches webpage content and detects meaningful changes based on keywords.

4

AI-powered interpretation

Gemini (Vertex AI API) analyzes detected changes to explain their importance and suggest next actions.

5

Take action

Important deadlines can be added directly to Google Calendar for timely reminders.
